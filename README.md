## Быстрый старт — Сборка ISO

### Требования (Ubuntu/Debian)

```bash
sudo apt-get install -y build-essential bc bison flex \
  libssl-dev libelf-dev libncurses-dev \
  cpio xz-utils gzip wget \
  grub-pc-bin grub-common xorriso isolinux
```

### Сборка

```bash
# 1. Клонируй или распакуй папку claudeos/
cd claudeos/

# 2. Дай права на исполнение
chmod +x build.sh

# 3. Запусти сборку (займёт 20-40 минут)
./build.sh

# 4. Готово! Файл claudeos.iso появится в папке
```

> ⚠️ **Внимание:** Сборка ядра Linux занимает 20-40 минут и требует ~3 GB свободного места.

## Настройка Limbo PC Emulator

### Установка Limbo

1. Скачай [Limbo PC Emulator](https://github.com/limboemu/limbo) из Google Play или GitHub
2. Установи на Android-устройство

### Конфигурация виртуальной машины

Открой Limbo → **New Machine** → настрой:

| Параметр | Значение |
|----------|----------|
| **Machine Type** | x86_64 |
| **CPU** | x86_64 |
| **CPU Cores** | 2 (или максимум) |
| **RAM** | 256 MB (рекомендуется 512 MB) |
| **CDROM** | `claudeos.iso` |
| **Hard Disk A** | None |
| **Boot from** | CDROM |
| **UI** | SDL / VNC |
| **Sound** | Отключить (для скорости) |

### Расположение ISO на Android

Скопируй `claudeos.iso` в папку:
```
/sdcard/limbo/
```
или
```
/storage/emulated/0/limbo/
```

### Файловый менеджер (claude-fm)

| Команда | Действие |
|---------|----------|
| `cd <dir>` | Перейти в директорию |
| `cp <src> <dst>` | Копировать |
| `mv <src> <dst>` | Переместить |
| `rm <file>` | Удалить |
| `mk <dir>` | Создать папку |
| `vi <file>` | Редактировать файл |
| `q` | Выйти |

### Текстовый редактор (claude-editor)

| Команда | Действие |
|---------|----------|
| `a` | Добавить строку в конец |
| `i N` | Вставить перед строкой N |
| `d N` | Удалить строку N |
| `r N` | Заменить строку N |
| `p` | Показать файл |
| `:w` | Сохранить |
| `:q` | Выйти |
| `:wq` | Сохранить и выйти |

### Терминал

В терминале доступны все команды BusyBox:
```bash
ls -la        # список файлов
cat file.txt  # просмотр файла
cp src dst    # копирование
mv src dst    # перемещение
rm file       # удаление
mkdir dir     # создать папку
grep text file # поиск
echo "text"   # вывод
free          # память
ps            # процессы
menu          # вернуться в меню
```

## Возможные проблемы

### "Kernel panic - not syncing"
→ Увеличь RAM до 512 MB в настройках Limbo

### Чёрный экран
→ Убедись что Boot from = CDROM  
→ Попробуй добавить `console=ttyS0,115200` в параметры ядра

### Очень медленная работа
→ Уменьши количество ядер до 1  
→ Отключи звук  
→ Используй SDL режим вместо VNC

### ISO не загружается
→ Проверь путь к файлу в Limbo  
→ Убедись что файл не повреждён: `md5sum claudeos.iso`

## Технические детали

- **Ядро:** Linux 6.1.x LTS (minconfig + virtio)  
- **Init система:** Минимальный sh-скрипт (PID 1)  
- **Libc:** Статическая сборка (нет внешних зависимостей)  
- **Размер ISO:** ~15-30 MB  
- **RAM при работе:** ~64-128 MB
